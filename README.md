# phpmvcproject

Install:
- Clone
- Make sure you have database file in config. E.g /config/database_mysql.php
- Have following tables in database:
 - comment
 - commentHandler
 - question
 - questionHandler
 - response
 - responseHandler
 - tag
 - tagHandler
 - user

live demo: http://www.student.bth.se/~fist14/phpmvc1/kmom10/webroot/
